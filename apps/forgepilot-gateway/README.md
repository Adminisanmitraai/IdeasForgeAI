# ForgePilot Cloud Gateway

FP-2A provides a browser-only gateway foundation on port `7071` using `express`, `ws`, and `cors`.

The UI shows:
- `ForgePilot Cloud Gateway`
- `Waiting for Desktop Companion...`
- `No desktop communication yet.`
- `No heartbeat yet.`
- `No pairing yet.`
- `No task relay yet.`
