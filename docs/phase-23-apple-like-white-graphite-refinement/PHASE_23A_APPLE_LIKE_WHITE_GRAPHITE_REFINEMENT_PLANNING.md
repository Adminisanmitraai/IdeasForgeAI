# Phase 23A — Apple-Like White/Graphite Visual Refinement Planning

Status: Completed, not frozen.

## Purpose

Phase 23A defines the final visual refinement plan to reduce the green-heavy look and make the IdeasForgeAI local main preview feel more Apple-like, clean, white, graphite, and premium.

This phase is planning only.

No frontend files are changed.
No main preview files are modified.
No deployment is performed.
No provider calls are added.
No database writes are added.
No Supabase/auth/secrets are added.
KisanMitraAI production is not touched.

## Current Issue

The current final frontend uses a soft AI green / mint palette.

It is working and validated, but visually it still feels too green.

## Desired Apple-Like Direction

Shift the visual system toward:

- white / off-white background
- graphite / black typography
- very light grey cards
- subtle shadows
- reduced green background wash
- emerald only for logo, CTA, and small accents
- calmer hero section
- more premium Apple-style spacing
- no loud gradients
- no heavy mint overlay

## Keep Unchanged

Do not change:
- page structure
- sections
- copy
- header layout
- CTA placement
- feature grid
- workflow section
- trust section
- final CTA
- footer
- read-only route
- validation routes
- deployment locks
- provider/database/secrets locks
