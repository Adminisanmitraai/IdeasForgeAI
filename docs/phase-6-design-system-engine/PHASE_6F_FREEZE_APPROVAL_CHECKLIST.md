# 4. Design Output Checks

The Design System Engine must produce:

## Design Positioning

One-line design direction.

## Brand Personality

What the product should feel like and what it should avoid.

## Visual Style

Light, clean, mobile-first visual rules.

## Typography Rules

Readable type hierarchy.

## Color Rules

Primary, background, surface, text, border, success, warning, and error usage.

## Layout Rules

Screen structure, spacing, card behavior, and mobile-first stacking.

## Component Rules

Reusable components and behavior.

## Interaction Rules

How users move forward, revise, approve, skip, or save.

## Mobile-First Rules

Narrow-screen behavior first.

## Accessibility Rules

Contrast, labels, focus states, and touch targets.

## Screen Guidance

Screen-by-screen design direction.

## Do-Not-Do Rules

Clear design and build boundaries.
