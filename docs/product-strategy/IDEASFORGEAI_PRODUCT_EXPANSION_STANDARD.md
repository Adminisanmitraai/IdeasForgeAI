# IdeasForgeAI Product Expansion Standard

IdeasForgeAI must become a universal AI builder for work, business, study, content creation, and daily life.

The platform should allow any user to describe their daily work, business need, academic requirement, content goal, or personal workflow in normal language. IdeasForgeAI should then create the correct output type: AI assistant app, website, dashboard, report, research document, presentation, catalog, proposal, inventory tool, sales tracker, accounting assistant, social media content pack, reel script pack, or household productivity assistant.

Supported users should include employees, bankers, accountants, retailers, restaurant owners, farmers, sales teams, creative agencies, students, researchers, teachers, doctors' admin teams, logistics teams, real estate workers, manufacturers, housewives, home business owners, content creators, Instagram/Reels creators, online sellers, and small business owners.

Every output must aim to be more polished, practical, professional, mobile-ready, approval-gated, and production-useful than a generic app builder result.

The platform must not only generate simple apps. It must understand sector, role, workflow, input data, expected output, safety level, approval requirement, and export format before generating the final result.

## Product Rule

IdeasForgeAI should help users create useful professional outputs without needing to learn AI, coding, automation, design, or document formatting.
